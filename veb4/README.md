# Лабораторная работа: Форма обратной связи на PHP

## Описание

Веб-приложение с формой обратной связи, реализующее обработку данных формы на PHP.

## Структура проекта

- `header.html` - HTML страница с формой обратной связи
- `index.php` - обработка формы и вывод формы
- `home.php` - обработка ответа и вывод результатов
- `uploads/` - папка для загруженных файлов (создается автоматически)

## Функционал формы

### Элементы формы:
- **ФИО** - однострочное текстовое поле
- **Ваш e-mail** - однострочное поле с placeholder
- **Сообщение** - многострочное текстовое поле
- **Тема обращения** - селектор (Предложение/Жалоба)
- **Откуда узнали** - две радио-кнопки:
  - Реклама из интернета
  - Рассказали друзья
- **Выбор файла** - поле для загрузки файла
- **Даю согласие на обработку данных** - флажок (не отмечен по умолчанию)
- **Кнопка "Отправить"**

### Обработка данных:

1. При отправке формы данные валидируются
2. Файл сохраняется в папку `uploads/`
3. Происходит переход на `home.php` с результатами

### Страница ответа (home.php):

Отображает:
- ФИО
- Текст обращения
- Название прикрепленного файла (если был)
- Кнопка "Заполнить снова"

### Возврат к форме:

При нажатии "Заполнить снова":
- Поля ФИО и Email заполняются ранее введенными значениями
- Выбирается ранее отмеченная радио-кнопка
- Выбирается ранее выбранная тема обращения

## Установка и запуск

### Вариант 1: XAMPP

1. Скопируйте папку `veb4` в `C:\xampp\htdocs\`
2. Откройте в браузере: `http://localhost/veb4/index.php`

### Вариант 2: OpenServer

1. Скопируйте папку `veb4` в `C:\OpenServer\domains\`
2. Откройте в браузере: `http://veb4/index.php`

### Вариант 3: PHP встроенный сервер

1. Откройте командную строку
2. Перейдите в папку: `cd "c:\Users\Ксенечка\Веб\veb4"`
3. Запустите: `php -S localhost:8000`
4. Откройте в браузере: `http://localhost:8000/index.php`

## Требования

- PHP 7.4 или выше
- Веб-сервер (Apache/Nginx) или PHP встроенный сервер
- Права на запись в папку `uploads/` (создается автоматически)

## Особенности реализации

- Все элементы формы выровнены по левому краю
- Подписи располагаются слева от элементов
- Одинаковый размер полей (кроме флажка и кнопки)
- Валидация данных на сервере
- Сохранение загруженных файлов
- Возврат к форме с заполненными данными
