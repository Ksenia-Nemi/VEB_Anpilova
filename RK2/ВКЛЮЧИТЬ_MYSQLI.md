# Как включить расширение MySQLi в PHP

## Способ 1: Через XAMPP (если используете XAMPP)

### Шаг 1: Найдите файл php.ini

Обычно файл `php.ini` находится в одной из следующих папок:
- `C:\xampp\php\php.ini`
- `C:\Program Files\xampp\php\php.ini`

### Шаг 2: Откройте php.ini в текстовом редакторе

**ВАЖНО:** Откройте файл от имени администратора (правой кнопкой → "Запуск от имени администратора")

### Шаг 3: Найдите и раскомментируйте строку

Найдите в файле строку:
```ini
;extension=mysqli
```

Или:
```ini
;extension=php_mysqli.dll
```

Уберите точку с запятой в начале строки, чтобы получилось:
```ini
extension=mysqli
```

Или:
```ini
extension=php_mysqli.dll
```

**Примечание:** В некоторых версиях PHP это может быть:
```ini
extension=mysqli
```
(без расширения .dll)

### Шаг 4: Сохраните файл

Сохраните изменения в файле.

### Шаг 5: Перезапустите Apache

1. Откройте **XAMPP Control Panel**
2. Нажмите **Stop** для Apache
3. Подождите несколько секунд
4. Нажмите **Start** для Apache

### Шаг 6: Проверьте, что расширение включено

Создайте файл `phpinfo.php` в папке `htdocs` со следующим содержимым:
```php
<?php
phpinfo();
?>
```

Откройте в браузере: `http://localhost/phpinfo.php`

Найдите раздел "mysqli" - он должен быть в списке. Если его нет, проверьте шаги выше.

## Способ 2: Если используете другой веб-сервер (OpenServer, Denwer и т.д.)

### Для OpenServer:
1. Откройте настройки OpenServer
2. Перейдите в "Настройки" → "Модули"
3. Убедитесь, что MySQLi включен
4. Перезапустите сервер

### Для Denwer:
1. Найдите файл `php.ini` в папке установки Denwer
2. Выполните шаги 2-4 из Способа 1
3. Перезапустите Denwer

## Способ 3: Проверка через командную строку

Откройте командную строку (cmd) и выполните:

```cmd
php -m
```

Это покажет список всех установленных модулей. Найдите в списке `mysqli`.

Если `mysqli` нет в списке, используйте Способ 1.

## Способ 4: Если расширение не устанавливается

Если после всех шагов расширение всё равно не работает:

1. **Проверьте путь к расширениям в php.ini:**

Найдите строку:
```ini
extension_dir = "ext"
```

Или:
```ini
extension_dir = "C:\xampp\php\ext"
```

Убедитесь, что путь правильный и указывает на папку `ext` внутри папки PHP.

2. **Проверьте, существует ли файл расширения:**

Проверьте, что файл `php_mysqli.dll` существует в папке `ext` (например, `C:\xampp\php\ext\php_mysqli.dll`)

3. **Проверьте версию PHP:**

Откройте командную строку и выполните:
```cmd
php -v
```

Убедитесь, что используется правильная версия PHP (XAMPP обычно использует свою версию PHP, а не системную).

## Дополнительная информация

- MySQLi (MySQL Improved Extension) обычно включено по умолчанию в современных версиях PHP
- Если вы используете PHP 8.0+, расширение может называться просто `mysqli` без `.dll`
- После любых изменений в `php.ini` необходимо перезапустить веб-сервер (Apache)

## Быстрая проверка

Создайте файл `test_mysqli.php` в корне проекта со следующим кодом:

```php
<?php
if (function_exists('mysqli_connect')) {
    echo 'MySQLi расширение УСТАНОВЛЕНО и работает!';
} else {
    echo 'MySQLi расширение НЕ установлено.';
}
phpinfo();
?>
```

Откройте этот файл в браузере. Если видите сообщение "MySQLi расширение УСТАНОВЛЕНО", значит всё работает правильно.
