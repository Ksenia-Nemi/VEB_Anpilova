# Интернет-магазин электроники

Проект интернет-магазина для рубежного контроля №2.

## Описание проекта

Современный интернет-магазин электроники с полным функционалом:
- Каталог товаров с фильтрацией
- Детальный просмотр товаров
- Система авторизации и регистрации
- Список покупок (wishlist)
- Форма обратной связи
- Адаптивный дизайн

## Технологии

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 7.4+
- **База данных:** MySQL
- **Сервер:** XAMPP (Apache + MySQL)

## Установка и настройка

### 1. Установка XAMPP

1. Скачайте и установите XAMPP с официального сайта: https://www.apachefriends.org/
2. Запустите Apache и MySQL через XAMPP Control Panel

### 2. Настройка базы данных

**Подробная инструкция находится в файле `ПОДКЛЮЧЕНИЕ_БД_XAMPP.md`**

Краткая версия:
1. Откройте phpMyAdmin: `http://localhost/phpmyadmin`
2. Создайте базу данных `shop_db` с кодировкой `utf8mb4_unicode_ci`
3. Импортируйте файл `database.sql` через вкладку "Импорт"

### 3. Настройка подключения

Файл `config.php` уже настроен для работы с XAMPP:
- Хост: `localhost`
- Пользователь: `root`
- Пароль: (пустой)
- База данных: `shop_db`

Если у вас другие настройки, отредактируйте `config.php`.

### 4. Размещение проекта

Поместите папку проекта в директорию:
```
C:\xampp\htdocs\RKR2\
```

### 5. Запуск

Откройте в браузере:
```
http://localhost/RKR2/
```

## Функционал

### Для всех пользователей:
- Просмотр каталога товаров
- Фильтрация по категориям
- Поиск товаров
- Переключение вида отображения (таблица/сетка)
- Просмотр детальной информации о товаре
- Слайд-шоу на главной странице

### Для авторизованных пользователей:
- Добавление товаров в список покупок
- Просмотр и управление списком покупок
- Отправка формы обратной связи

## Тестовые данные

После импорта базы данных доступны:
- **Тестовый пользователь:**
  - Логин: `admin`
  - Пароль: `admin123`

- **Товары:** 8 товаров в 3 категориях (Смартфоны, Ноутбуки, Планшеты)

## Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache (входит в XAMPP)
- Браузер с поддержкой современного JavaScript

## Особенности реализации

- Использование подготовленных SQL запросов для безопасности
- Хеширование паролей через `password_hash()`
- Адаптивный дизайн для мобильных устройств
- Слайд-шоу с автоматической сменой слайдов
- Сохранение предпочтений пользователя (вид отображения товаров)
- Валидация форм на стороне сервера
